Explicação do Web Programa

  • ✔ Passo 1: Esse web programa foi desenvolvido com as linguagens de programação: HTML, CSS, JAVASCRIPT e foi utilizado uma pastas tabela como diretório para armazenar as tabelas referentes aos dados fictícios de imóveis dos anos de 2020 até 2026.
  • ✔ Passo 2: O objetivo do programa é possibilitar um sistema rápido e prático para a consulta de metragens edificadas ou não para a finalidade de elaboração de certidões narrativas. A princípio propronho que as tabelas de dados de cada ano podem ser construídas com os arquivos da emissão dos carnes de IPTU de cada ano, ou seja, esse seria um banco de dados para o programa web de consulta.
  • ✔ Passo 3: O sistema deverá ser consultado pelos usuários de duas formas incialmente pelas inscrições com o final "000" e depois com o final com alguma númeração que já indiquem área edificada, por exemplo "001".
  • ✔ Passo 4: A consulta individualizada por inscrição irá trazer as respectivas informações: inscrição municipal, nº da quadra, nº do lote, ano, netragem, utilização e estrutura.
  • ✔ Passo 5: Observo que a consulta nesse sistema não dispensa apuração nos sistemas internos da Prefeita para a finalidade de apuração se o contribuinte requereu alguma revisão histórica de IPTU que tangem a metragem.
  • ✔ Passo 6: Como exemplo coloco alguns exemplo de inscrições para teste de consulta: 1º caso: 08.777.141.0012.000 (que é um terreno vago); 2º caso 08.777.143.0012.998 (que continha edificação residencial por um período e depois se transformou numa edificação industrial) e o 3º caso 08.777.143.0024.999 (que trata-se de uma edificação que surgi apenas no ano de 2026).
  • ✔ Passo 7: O Web Programa Digital é um fruto da extensão acadêmica do 7º semestre de Engenharia de Software da universidade Anhanguera no primeiro semestre de 2026. Ele foi projetado para otimizar o fluxo de trabalho da repartição, substituindo consultas manuais lentas por uma interface ágil e centralizada e com lógica de segurança para inibir que terceiros tenham acesso as informações. Este sistema transforma dados brutos em informações acessíveis, permitindo que o servidor público tome decisões técnicas com muito mais rapidez e precisão.
